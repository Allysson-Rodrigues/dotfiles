# 💻 Dotfiles | High-Performance Zsh Environment

Personal configuration files optimized for **Node.js** development, **Docker** management, and professional workflow. Designed for **WSL2** and **GitHub Codespaces**.

## 🚀 Key Features

- **Zinit Package Manager**: Fast and asynchronous plugin loading to keep the terminal responsive.
- **Spaceship Prompt**: A clean, information-rich prompt displaying Git status and Node.js versions.
- **Productivity Aliases**: Custom shortcuts for Git, Docker, and Node.js to speed up the daily workflow.
- **Automated Setup**: A dedicated `install.sh` script to replicate the environment in seconds.

## 🛠️ Included Plugins

- `zsh-users/zsh-autosuggestions`: Intelligent command history suggestions.
- `zdharma-continuum/fast-syntax-highlighting`: Real-time syntax highlighting for shell commands.
- `zsh-users/zsh-completions`: Enhanced tab-completion for various tools.

## 📦 Installation

To apply these configurations to your environment, run:

```bash
git clone [https://github.com/Allysson-Rodrigues/dotfiles.git](https://github.com/Allysson-Rodrigues/dotfiles.git) ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```
